Authentication

A collection of Authentication specific endpoints.

POST api/v1/login

範例請求:
curl --request POST \
    "http://127.0.0.1:8000/api/v1/login" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"email\": \"admin@admin.com\",
    \"password\": \"super-secret-password\"
}"

const url = new URL(
    "http://127.0.0.1:8000/api/v1/login"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "admin@admin.com",
    "password": "super-secret-password"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
接收回應:
請求失敗並出現錯誤:


提示：檢查您是否正確連接到網路。
如果您是此 API 的維護者，請驗證您的 API 正在運作並且您已啟用跨來源資源共享。
您可以檢查開發人員工具主控台以取得偵錯資訊。

請求      

POST api/v1/login

標頭

Content-Type      

Example: application/json

Accept      

Example: application/json

內文參數

email   string   

認證使用者的電子郵件。. :Attribute 必須是有效的 E-mail。. Example: admin@admin.com

password   string   

認證使用者的密碼。. Example: super-secret-password

請求      

POST api/v1/register

標頭

Content-Type      

Example: application/json

Accept      

Example: application/json

內文參數

name   string   

註冊使用者的姓名。. :Attribute 不能小於 2 個字元。 :Attribute 不能多於 255 個字元。. Example: 系統管理員

email   string   

註冊使用者的電子郵件。. :Attribute 必須是有效的 E-mail。 :Attribute 不能多於 255 個字元。. Example: admin@admin.com

password   string   

註冊使用者的密碼。. :Attribute 不能小於 8 個字元。. Example: super-secret-password
